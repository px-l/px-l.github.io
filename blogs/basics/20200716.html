不同的二叉搜索树

vuePress-theme-reco 刘冲    2020

不同的二叉搜索树

刘冲 2020-07-16 每日一算 js LeetCode

# 题目描述：

给定一个无向图graph，当这个图为二分图时返回true。

如果我们能将一个图的节点集合分割成两个独立的子集A和B，并使图中的每一条边的两个节点一个来自A集合，一个来自B集合，我们就将这个图称为二分图。

graph将会以邻接表方式给出，graph[i]表示图中与节点i相连的所有节点。每个节点都是一个在0到graph.length-1之间的整数。这图中没有自环和平行边： graph[i] 中不存在i，并且graph[i]中没有重复的值。 ————————————————

# 解法

对于图中的任意两个节点 u 和 v，如果它们之间有一条边直接相连，那么 u 和 v 必须属于不同的集合。

如果给定的无向图连通，那么我们就可以任选一个节点开始，给它染成蓝色。随后我们对整个图进行遍历，将该节点直接相连的所有节点染成红色，表示这些节点不能与起始节点属于同一个集合。我们再将这些红色节点直接相连的所有节点染成红色，以此类推，直到无向图中的每个节点均被染色。

如果我们能够成功染色，那么红色和蓝色的节点各属于一个集合，这个无向图就是一个二分图；如果我们未能成功染色，即在染色的过程中，某一时刻访问到了一个已经染色的节点，并且它的颜色与我们将要给它染上的颜色不相同，也就说明这个无向图不是一个二分图。

/**
 * @param {number[][]} graph
 * @return {boolean}
 */
const isBipartite = (graph) => {
    let len = graph.length;
    let visited = new Array(len);
    for(let i = 0;i<len;i++){
        if(visited[i]){
            continue
        }
        visited[i] = 1;//如果是没有相连的节点置为蓝色
        let queue=[i];//当前已染色顶
        while(queue.length){
            let cur = queue.shift();
            let arr = graph[cur];
            let neighborColor = -visited[cur];
            for(let j=0,lenJ=arr.length;j<lenJ;j++){
                let item = arr[j]; 
                if(!visited[item]){
                    visited[item] = neighborColor;
                    queue.push(item)
                }else if(visited[item]&&visited[item]!==neighborColor){
                    return false
                }
            } 
        }
        
    }
    return true;
};
